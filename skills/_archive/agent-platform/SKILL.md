# Agent Platform Skill

Auto-generated from ProjectD session: 2026-04-11 — COO delegation + SSE routing + dynamic agent creation

## Core Pattern: COO Orchestration

The "Chief Operating Officer" pattern: one primary orchestrator agent delegates to specialist agents. The COO:

1. Receives user messages
2. Decides which specialist to delegate to (researcher, writer, ops, or dynamically created agents)
3. Calls `delegate(agent, task)` — a skill/tool that routes to the specialist
4. Shows the specialist's full response in THEIR chat channel (via SSE), not the COO's
5. Returns a short summary/confirmation to the COO's channel

## Step 1: Define the COO Persona

Key persona elements for free/open models (critical — see tech_insight_free_model_tool_calling):

- "IMMEDIATELY call the function. Do NOT describe what you would do."
- "When the user says X, call `delegate('researcher', '...')`. Do not explain. Just call it."
- List each tool with exact trigger → action mapping
- Remove all hedging language ("I would", "I can", "I'll prepare")

  const { toAgent, task } = JSON.parse(e.data);
  appendMessage(toAgent + "-chat", "user", task);
});
sse.addEventListener("delegation:response", (e) => {
  const { toAgent, response } = JSON.parse(e.data);
  appendMessage(toAgent + "-chat", "assistant", response);
});
sse.addEventListener("agent:created", (e) => {
  // Add new agent tab to sidebar without page reload

});

## Step 5: Safari compatibility fixes

Always apply to any web API serving dynamic data:

- `Cache-Control: no-store` on all `/api/` and `/auth/` routes (Safari caches aggressively)
- CDN scripts: use `async` attribute; verify version exists before pinning
- Sort default agent (COO/is_default) first in agents SQL: `ORDER BY is_default DESC`

## Common Pitfalls

- **DB column mismatch:** Nightly scripts may add column checks before migrations run. Always query `information_schema.columns` to verify before using new columns.
- **Free model tool calling:** Smaller models describe tools instead of calling them. Fix with imperative persona (see Step 1).
- **CDN 404:** Always verify CDN version URLs with `curl -I` before committing.
