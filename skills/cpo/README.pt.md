# Chief Product Officer AI

## O que faz

Orquestra ciclos de vida completos de produtos, desde a ideia até a produção. Transforma conceitos brutos em aplicações prontas para produção através de descoberta estruturada, planejamento estratégico, implementação iterativa por estágios e testes rigorosos. Usa subagentes especializados (pesquisa de produto, arquitetura, design, API, banco de dados, deployment) para entregar soluções de classe mundial com documentação completa.

## Como invocar

```bash
/cpo-go <nome-projeto> <descrição>

# Exemplos:
/cpo-go game criar um jogo interativo tic-tac-toe
/cpo-go taskflow construir app de gerenciamento de tarefas para pequenos times
/cpo-go artmarket criar marketplace onde artistas vendem arte digital
```

**Triggers adicionais:** `build this product`, `cpo mode`, `chief product officer`, `product lifecycle`, `idea to production`, `full product build`, `strategic planning`, `product roadmap`

## Quando usar

- Transformar uma ideia em produto completo com arquitetura profissional
- Executar planejamento de produto com épicos, estágios e histórias validadas
- Implementar aplicações multi-camadas (UI, API, banco de dados) em paralelo
- Garantir qualidade com testes e validação end-to-end antes do deploy
- Gerar documentação de usuário e técnica para go-live
