# Firecrawl - Extração de Dados Web

## O que faz

Extrai, rastreia e coleta dados de qualquer site usando a plataforma Firecrawl. Converte páginas web em markdown pronto para LLM, com suporte a renderização JavaScript, bypass de anti-bot e rotação de proxies. Ideal para web scraping, crawling de sites completos, busca na web com extração de resultados e pesquisa autônoma multi-fonte.

## Como invocar

```
/firecrawl [tarefa]
```

**Exemplos:**
- `/firecrawl Extraia o conteúdo de https://example.com como markdown`
- `/firecrawl Rastreie todo o site https://docs.example.com e organize por seção`
- `/firecrawl Busque "ferramentas de gestão de projetos" e extraia preços e features`
- `/firecrawl Analise os concorrentes: preços em site-a.com, site-b.com e site-c.com`

## Quando usar

- **Análise competitiva**: comparar preços, features e posicionamento de concorrentes
- **Pesquisa de mercado**: coletar dados de múltiplas fontes web de forma automatizada
- **Indexação de documentação**: extrair conteúdo completo de sites de docs ou wikis
- **Inteligência corporativa**: pesquisa autônoma sobre empresas (funding, team, revenue)
